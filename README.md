Shopping Website (PHP + MySQL)

A fully functional e-commerce website built using PHP, MySQL, AJAX, and jQuery with separate modules for both customers and admins.  
This project includes user authentication, cart management, order flow, coupon system, and an admin dashboard for product & category maintenance.

---

Live Demo

https://shoppingsite.iceiy.com/

---

Features

Customer Side
- User sign-up and login  
- Product browsing  
- Add to cart / remove from cart  
- Apply coupons  
- Checkout and order placement  
- View placed orders  

Admin Panel
- Add / delete products  
- Add / edit / delete categories  
- Manage orders  
- Block / unblock users  
- Dashboard-style access  

Technical Features
- AJAX-based dynamic updates (no page reloads)  
- Secure login system  
- Structured backend flow  
- Reusable PHP components  

---

Tech Stack
- Backend: PHP  
- Database: MySQL  
- Frontend: HTML, CSS, JavaScript, jQuery  
- Server: XAMPP / Apache  
- Tools: phpMyAdmin  


---

How to Run Locally

1. Download or clone the repository  
2. Create a database in phpMyAdmin  
3. Import the SQL file from `/database`  
4. Update `connection.php` with your own DB credentials  
5. Run the project using XAMPP (Apache + MySQL)  
6. Open `http://localhost/<your-folder-name>` in your browser  

---

Important
Before using this project in production, make sure to:
- Configure secure passwords  
- Use prepared statements  
- Disable error display in production
